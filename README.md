# 🦷 Sistema Odontológico Web (Seminário Tech - UNITINS)

Sistema web desenvolvido para gerenciamento de atendimentos odontológicos em postos de saúde, na rede pública de Paranã Tocantins.

# Link do Protótipo no Figma:
>> https://www.figma.com/proto/12vn15cWCxZvUodBcEBCcx/Prot%C3%B3tipo-Semin%C3%A1rio-Tech?node-id=0-1&t=8tOzfw6g0TlUwlla-1

---
# O projeto possui:

- Tela de Login
- Tela de Recuperação de Senha
- Dashboard público com chamada de senhas por voz
- Gestão de pacientes
- Gestão de atendimentos
- Cadastro de pacientes
- Novo atendimento
- Detalhes do paciente
- Detalhes do atendimento
- Receituário
- Relatórios
- Laudos
- Menu lateral universal
- Responsividade
- Simulação de fila de atendimento
